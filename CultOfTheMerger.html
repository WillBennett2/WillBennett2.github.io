About the project

This project's aim was to create and design a mobile game that is able to be released. This project was dedicated to the whole year and along side to the game, documentation had to be created. This included a technical design document which went over the code base of the game and a game design document which explored and set out any design decisions that were made. The project needed a vertical and horizontal prototype to be made. This game was made for Android by using Unity.

The gameplay loop of this project follows a typical idle game where the player has active and inactive periods of gameplay. Within the active period the player is able to buy buildings, spawn minions, and expand their cult.
